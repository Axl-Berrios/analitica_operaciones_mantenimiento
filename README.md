# analitica_operaciones_mantenimiento
El informe analiza la gestión de operaciones y mantenimiento mediante analítica de datos, integrando información operativa y financiera para evaluar costos, eficiencia y cumplimiento presupuestario. Los resultados permiten optimizar la planificación, reducir el gasto en un 12% y fortalecer el control de gestión y la toma de decisiones estratégicas
